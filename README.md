# News Spy
Chatbot espião de notícias que vigia notícias de seu interesse e as envia em ordem de relevância

![Banner do News Spy](docs/assets/banner.png)

## Frequências possíveis

- De hora em hora (`every-hour`)
- Diária (`every-day`)
- Semanal (`every-week`)

default: `every-hour`
